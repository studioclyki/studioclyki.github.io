Informações Coletadas

Informações fornecidas pelo usuário: Podemos coletar informações como nome de usuário, e-mail e outros dados fornecidos voluntariamente para personalização e comunicação.

Anúncios

Informações de uso: Utilizamos o AdMob do Google para exibir anúncios. Dados de interação com anúncios são coletados de forma anônima.

Informações do dispositivo: Podemos coletar dados técnicos como modelo, sistema operacional e identificadores, de forma anônima, para melhorar o desempenho.

Uso das Informações

Utilizamos as informações para melhorar nossos aplicativos, desenvolver novas funcionalidades e exibir anúncios mais relevantes.

Compartilhamento de Informações

Não compartilhamos dados pessoais identificáveis, exceto quando exigido por lei. Dados anônimos podem ser usados para análises e melhorias.

Anúncios Comportamentais

Alguns anúncios podem ser baseados nos interesses do usuário, utilizando cookies ou tecnologias similares, sempre de forma anônima.

Configurações de Anúncios

Você pode desativar anúncios personalizados nas configurações do seu dispositivo Google.

Segurança

Adotamos medidas para proteger seus dados, porém nenhum sistema é 100% seguro.

Alterações

Esta política pode ser atualizada a qualquer momento. Recomendamos revisá-la periodicamente.
